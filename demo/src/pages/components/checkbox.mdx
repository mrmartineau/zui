# Checkbox

Custom-styled checkboxes with list layout.

## Usage

<Demo code={`<label class="zui-checkbox">
  <input type="checkbox" checked />
  Accept terms
</label>`}>
  <label class="zui-checkbox">
    <input type="checkbox" checked />
    Accept terms
  </label>
</Demo>

<CodeTabs
  html={`<label class="zui-checkbox">
  <input type="checkbox" checked />
  Accept terms
</label>`}
  react={`<label className="zui-checkbox">
  <input type="checkbox" defaultChecked />
  Accept terms
</label>`}
  astro={`<label class="zui-checkbox">
  <input type="checkbox" checked />
  Accept terms
</label>`}
/>
