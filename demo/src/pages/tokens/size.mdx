## Viewport units

```css
--size-svw: 100svw;
--size-svh: 100svh;
--size-lvw: 100lvw;
--size-lvh: 100lvh;
--size-dvw: 100dvw;
--size-dvh: 100dvh;
```

## Intrinsic sizing

```css
--size-min: min-content;
--size-max: max-content;
--size-fit: fit-content;
```

## Usage

```css
.sidebar {
  width: var(--size-quarter);
}

.hero {
  min-height: var(--size-dvh);
}
```
