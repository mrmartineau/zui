## Install

```sh
npm install your-package
```

## Use it

Document your package's API here. Write standard Markdown, drop in code blocks,
and they'll be syntax-highlighted automatically.

```ts
import { thing } from 'your-package'

thing()
```

## Add a new page

Create a new `.mdx` file anywhere under `src/pages/<section>/` and it shows up in
the sidebar automatically. For example, `src/pages/guides/configuration.mdx`
adds a "Configuration" entry to the Guides section.
