## How this site works

- Every `.mdx` file under `src/pages/<section>/` becomes a sidebar entry.
- Each section's `index.mdx` sets the section's `sectionOrder` (and appears at
  the top of its sidebar group).
- Set `order` in a page's frontmatter to control its position in the sidebar.
- Edit `src/site.config.ts` to change the site name, links, and version.
