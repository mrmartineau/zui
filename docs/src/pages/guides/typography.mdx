# Typography

Fluid type scale, font stacks, and typography utility classes powered by design tokens.

## Type Scale

ZUI uses a fluid type scale generated with <a href="https://utopia.fyi" class="zui-link">Utopia</a>. Sizes interpolate between viewport widths for seamless responsive typography.

## Usage

```html
/* With utility classes */
<p class="zui-text-2">Large text</p>
<p class="zui-text-base">Base text</p>
<p class="zui-text--1">Small text</p>
```

```css
/* With custom properties */
.my-heading {
  font-size: var(--step-4);
}
```

## Font Stacks

Configure fonts in <code class="zui-code">theme.css</code>. ZUI provides robust fallback stacks.

<TokenGrid>
  <TokenRow>
    <CopyCode>--font-sans</CopyCode>
    <span style={{ fontFamily: 'var(--font-sans)' }}>The quick brown fox jumps over the lazy dog</span>
  </TokenRow>
  <TokenRow>
    <CopyCode>--font-serif</CopyCode>
    <span style={{ fontFamily: 'var(--font-serif)' }}>The quick brown fox jumps over the lazy dog</span>
  </TokenRow>
  <TokenRow>
    <CopyCode>--font-mono</CopyCode>
    <span style={{ fontFamily: 'var(--font-mono)' }}>The quick brown fox jumps over the lazy dog</span>
  </TokenRow>
</TokenGrid>

```css
/* Override in theme.css */
:root {
  --custom-font-sans: Geist, Inter;
  --custom-font-serif: Times New Roman;
  --custom-font-mono: Geist Mono;
}
```
