## Customising shadow colour

Override <code class="zui-code">--shadow-color</code> in your theme to tint all shadows. It defaults to a neutral near-black (<code class="zui-code">oklch(0.2 0 0)</code>) and is overridden to <code class="zui-code">--color-sky-600</code> in the default theme.

```css
:root {
  /* Neutral shadows */
  --shadow-color: oklch(0.2 0 0);

  /* Tinted shadows */
  --shadow-color: var(--color-indigo-600);
}
```

## Usage

```css
.card {
  box-shadow: var(--shadow-md);
}

.modal {
  box-shadow: var(--shadow-xl);
}
```
