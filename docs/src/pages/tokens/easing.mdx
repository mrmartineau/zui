## Aliases

```css
--ease-elastic: var(--ease-elastic-out);
--ease-squish: var(--ease-elastic-in-out);
```

## Usage

```css
.button {
  transition: background-color 200ms var(--ease-in-out);
}

.modal {
  animation: slide-in-up 0.4s var(--ease-spring);
}
```
