`Kbd` renders a single keyboard key. Wrap several in `KbdGroup` to show a shortcut such as <kbd class="zui-kbd-group"><kbd class="zui-kbd">⌘</kbd><kbd class="zui-kbd">K</kbd></kbd>. Both render a `<kbd>` element — nesting keys inside a group `<kbd>` is valid HTML and conveys "composite input".

## Usage

<Demo
  html={`<kbd class="zui-kbd">Esc</kbd>`}
  react={`import { Kbd } from '@mrmartineau/zui/react'

<Kbd>Esc</Kbd>`}
  astro={`---
import { Kbd } from '@mrmartineau/zui/astro'
---

<Kbd>Esc</Kbd>`}
  solid={`import { Kbd } from '@mrmartineau/zui/solid'

<Kbd>Esc</Kbd>`}
  vue={`<template>
  <Kbd>Esc</Kbd>
</template>

<script setup>
import { Kbd } from '@mrmartineau/zui/vue'
</script>`}
  svelte={`<script lang="ts">
  import { Kbd } from '@mrmartineau/zui/svelte'
</script>

<Kbd>Esc</Kbd>`}
>
  <kbd class="zui-kbd">Esc</kbd>
</Demo>

## CSS custom properties

Set on `.zui-kbd`.

| Property | Default | Description |
|---|---|---|
| `--zui-kbd-bg` | `var(--color-surface)` | Key background |
| `--zui-kbd-color` | `var(--color-text)` | Key label colour |
| `--zui-kbd-border-color` | `var(--color-border)` | Border + drop-shadow colour |
| `--zui-kbd-radius` | `var(--radius-md)` | Border radius |
| `--zui-kbd-font-size` | `var(--step--2)` | Label font size |
