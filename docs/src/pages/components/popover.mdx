# Popover

Floating panels that appear below their trigger, built on the native [HTML Popover API](https://developer.mozilla.org/en-US/docs/Web/API/Popover_API) and [CSS Anchor Positioning](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_anchor_positioning). No JavaScript required.

The trigger needs `anchor-name` set to match the popover's `id` (prefixed with `--`). The Astro and React wrappers set `position-anchor` automatically from the `id` prop.

## Usage

<Demo code={`<button
  class="zui-button"
  popovertarget="demo-popover"
  style="anchor-name: --demo-popover"
>
  Open popover
</button>
<div id="demo-popover" popover class="zui-popover" style="position-anchor: --demo-popover;">
  <p>Popover content goes here.</p>
</div>`}>

  </div>
</Demo>

## Auto vs manual

`popover="auto"` (default) closes when clicking outside or pressing <kbd>Esc</kbd>. `popover="manual"` requires an explicit close trigger.

<Demo code={`<button class="zui-button" popovertarget="pop-auto" style="anchor-name: --pop-auto">
  Auto (click outside to close)
</button>
<div id="pop-auto" popover="auto" class="zui-popover" style="position-anchor: --pop-auto;">
  <p>I close when you click outside.</p>
</div>

<button class="zui-button zui-button-variant-outline" popovertarget="pop-manual" style="anchor-name: --pop-manual">
  Manual (must use close button)
</button>
<div id="pop-manual" popover="manual" class="zui-popover" style="position-anchor: --pop-manual;">
  <p>I stay open until explicitly closed.</p>
  <button class="zui-button zui-button-size-sm" style="margin-top:0.5rem;" popovertarget="pop-manual" popovertargetaction="hide">
    Close
  </button>
</div>`}>
  <div style="display:flex;gap:1rem;flex-wrap:wrap;">
    <button class="zui-button" popovertarget="pop-auto-demo" style="anchor-name: --pop-auto-demo">
      Auto (click outside to close)
    </button>
    <div id="pop-auto-demo" popover="auto" class="zui-popover" style="position-anchor: --pop-auto-demo;">
      <p>I close when you click outside.</p>
    </div>
    <button class="zui-button zui-button-variant-outline" popovertarget="pop-manual-demo" style="anchor-name: --pop-manual-demo">
      Manual (must use close button)
    </button>
    <div id="pop-manual-demo" popover="manual" class="zui-popover" style="position-anchor: --pop-manual-demo;">
      <p>I stay open until explicitly closed.</p>
      <button class="zui-button zui-button-size-sm" style="margin-top:0.5rem;" popovertarget="pop-manual-demo" popovertargetaction="hide">
        Close
      </button>
    </div>
  </div>
</Demo>
