# Menu

A dropdown menu built on the native `<details>`/`<summary>` elements — no JavaScript required. The menu panel uses `::details-content` for absolute positioning.

## Usage

Menu items inherit all styles from the [Button](/components/button) component. Always combine `.zui-menu-item` with `.zui-button` and a variant class (typically `.zui-button-variant-ghost`). The summary trigger similarly uses `.zui-button` classes.

## CSS custom properties

These custom properties are set on `.zui-menu[open]::details-content` and can be overridden inline or in your own CSS.

| Property | Default | Description |
|---|---|---|
| `--zui-menu-radius` | `var(--radius-xl)` | Border radius of the dropdown panel |
| `--zui-menu-border` | `var(--color-border)` | Border colour |
| `--zui-menu-bg` | `var(--color-surface)` | Background colour |
| `--zui-menu-padding` | `var(--space-2xs)` | Padding inside the panel |
