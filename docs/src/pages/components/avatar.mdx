# Avatar

A circular (or shaped) image with a fallback for when the image fails to load or no source is provided.

## Usage

<Demo code={`<span class="zui-avatar">
    <span class="zui-avatar-fallback"><i class="ph ph-user" aria-hidden="true"></i></span>
    <img
      class="zui-avatar-image"
      src="/avatar.webp"
      alt="User"
      onerror="this.setAttribute('data-error', '')"
    />
</span>`}>
  <span class="zui-avatar">
    <span class="zui-avatar-fallback"><i class="ph ph-user" aria-hidden="true"></i></span>
    <img class="zui-avatar-image" src="/avatar.webp" alt="User" onerror="this.setAttribute('data-error', '')" />
  </span>
</Demo>

<CodeTabs
  html={`<span class="zui-avatar">
  <span class="zui-avatar-fallback"><i class="ph ph-user" aria-hidden="true"></i></span>
  <img
      class="zui-avatar-image"
      src="/avatar.webp"
      alt="User"
      onerror="this.setAttribute('data-error', '')"
    />
</span>`}
  react={`import { Avatar } from '@mrmartineau/zui/react'

<Avatar src="/avatar.webp" alt="User" />`}
  astro={`---
import { Avatar } from '@mrmartineau/zui/astro'
---

<Avatar src="/avatar.webp" alt="User" />`}
/>

## Fallback

When no `src` is provided or the image fails to load, the fallback content is shown. If no fallback is specified, a default user icon is displayed.

### Default icon fallback

<Demo code={`<span class="zui-avatar">
  <span class="zui-avatar-fallback"><i class="ph ph-user" aria-hidden="true"></i></span>
</span>`}>
  <span class="zui-avatar">
    <span class="zui-avatar-fallback"><i class="ph ph-user" aria-hidden="true"></i></span>
  </span>
</Demo>

### Initials fallback

<Demo code={`<span class="zui-avatar">
  <span class="zui-avatar-fallback">AB</span>
</span>`}>
  <span class="zui-avatar">
    <span class="zui-avatar-fallback">AB</span>
  </span>
</Demo>

<CodeTabs
  html={`<!-- Default icon -->
<span class="zui-avatar">
  <span class="zui-avatar-fallback"><i class="ph ph-user" aria-hidden="true"></i></span>
</span>

<!-- Initials -->
<span class="zui-avatar">
  <span class="zui-avatar-fallback">AB</span>
</span>`}
  react={`import { Avatar } from '@mrmartineau/zui/react'

{/* Default icon */}
<Avatar />

{/* Initials */}
<Avatar fallback="AB" />`}
  astro={`---
import { Avatar } from '@mrmartineau/zui/astro'
---

{/* Default icon */}
<Avatar />

{/* Initials */}
<Avatar fallback="AB" />`}
/>
